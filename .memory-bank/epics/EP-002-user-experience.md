# EP-002: User Experience & Methodology Value

## Business value
Deliver the core methodology value — blind spot detection, actionable output, and smooth onboarding. This is what makes users come back: they get real insight from the tool.

## Scope
- Interactive onboarding tour (rework existing overlay)
- Decision memo export (Markdown)
- Blind spot / gap analysis UX
- TODO cards as action plan
- Mobile UX improvements (P2)

## Features
- FT-003: Interactive onboarding tour
- FT-004: Decision memo export
- FT-005: Blind spot analysis
- FT-007: TODO cards panel
- FT-015: Mobile UX improvements (P2)

## Success metrics
- Users complete first session without confusion (measure via Webvisor)
- >= 30% of completed sessions produce a decision memo export
- Average session duration 15-30 min
- >= 50 completed sessions (>5 cards)
