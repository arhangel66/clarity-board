# FT-009: Demand Validation

## Goal
Validate product-market fit signal with real usage data.

## Acceptance criteria
- [ ] 5-10 own sessions on different use cases completed
- [ ] 5-10 external users recruited and observed
- [ ] Webvisor recordings analyzed for UX issues
- [ ] Funnel data from FT-002 reviewed
- [ ] Findings documented with go/no-go decision

## Notes
This is a process/research feature, not code. Depends on analytics (FT-002) being live.
